# Aviation Incident Sequence Modeling

This project analyzes aviation incident data from the NTSB database to
understand event sequences and build models that predict incident
severity based on early-stage signals.

------------------------------------------------------------------------

## Project Structure

    project_root/

    ├── notebooks/

    │
    ├── data/
    │   ├── avall.mdb
    │   └── csv_outputs/
    │
    └── README.md

------------------------------------------------------------------------

## Overview

The dataset is sourced from the NTSB aviation database (.mdb format).
This project:

-   Extracts structured tables using mdb-export\
-   Converts raw tables into CSV format\
-   Performs exploratory data analysis (EDA)\
-   Constructs event sequences for each incident\
-   Builds models to predict severity from early event signals

## Setup Instructions

### 1. Clone the repository

    git clone <your-repo-url>
    cd <your-repo-name>

### 2. Install dependencies

    pip install pandas numpy matplotlib seaborn scikit-learn tensorflow xgboost networkx

### 3. Install mdbtools

Mac:

    brew install mdbtools

Linux:

    sudo apt-get install mdbtools
